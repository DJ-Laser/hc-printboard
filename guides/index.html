PrintBoard Getting Started:

First, Think of a idea!

To build something, you need an idea! Think about what you would like to display on your pegboard, once you get one. What is something you always lose and need a place to store? What do you think would look really cool hung up on a wall?

Next, set up your CAD software of choice

First, choose and download your software (or sign up if your using a web based one), then install wakatime for your platform of choice Wakatime for most desktop CAD softwares: Wakatime AnyIDE Wakatime for onshape: Onshape Wakatime

Start Building!

Now that you have the project set up, now you just need to start building! if you have never used CAD before, I recommend onshape. A tutorial for onshape can be found below
